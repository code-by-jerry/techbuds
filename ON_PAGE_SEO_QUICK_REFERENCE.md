# On-Page SEO Quick Reference Guide

## ✅ Heading Structure Rules

### H1 (Page Title)
- **Only ONE H1 per page**
- Contains primary keyword
- Should be the main page title
- 50-70 characters ideal

### H2 (Main Sections)
- Used for main content sections
- Should follow H1 logically
- Can have multiple H2s
- Contains secondary keywords

### H3 (Sub-sections)
- Used for sub-sections under H2
- Supports H2 content
- Can have multiple H3s
- More specific topics

## 📋 Current Page Structure

### Homepage
- H1: "Techbuds - Design Develop Deliver | Web & Mobile App Development Services in India" (visually hidden)
- H2: Hero carousel headings (multiple)
- H2: "Comprehensive digital solutions for your business"
- H2: "What Our Clients Say"
- H3: Service sub-sections

### Service Pages
- H1: Service-specific title (e.g., "Fast, Scalable & SEO-Optimized Websites")
- H2: "Why Web Development Matters for Business Growth"
- H2: "Our Web Development Services"
- H2: "Technologies We Use"
- H3: Individual service features

### Blog Posts
- H1: Blog post title
- H2: Content section headings
- H3: Sub-headings within sections

## 🎯 Image Alt Text Best Practices

### Format:
- Descriptive + Context + Brand
- Example: "Web Development Services - Fast, Scalable Websites by Techbuds"

### Rules:
- ✅ Descriptive and specific
- ✅ Include relevant keywords naturally
- ✅ Add context (what, where, why)
- ✅ Keep it concise (125 characters max)
- ❌ No keyword stuffing
- ❌ No "image of" or "picture of"

## 📝 Title Tag Optimization

### Format:
- Primary Keyword | Brand Name
- Or: Brand Name - Primary Keyword

### Rules:
- 50-60 characters (optimal)
- Unique per page
- Primary keyword in first 60 chars
- Include brand name
- Compelling and descriptive

## 📄 Meta Description Optimization

### Format:
- Value proposition + Call-to-action

### Rules:
- 150-160 characters
- Unique per page
- Include primary keyword naturally
- Compelling and action-oriented
- Clear value proposition

## 🔗 Internal Linking Best Practices

### Anchor Text:
- Descriptive and keyword-rich
- Natural in context
- Not just "click here" or "read more"

### Placement:
- Within content (natural context)
- Related content sections
- Breadcrumb navigation
- Footer links

## ✅ Checklist for Every Page

- [ ] Single H1 tag
- [ ] Proper H1-H2-H3 hierarchy
- [ ] Title tag optimized (50-60 chars)
- [ ] Meta description optimized (150-160 chars)
- [ ] All images have descriptive alt text
- [ ] Content is 300+ words (ideally 1000+)
- [ ] Keywords used naturally (1-2% density)
- [ ] Internal links with descriptive anchor text
- [ ] Breadcrumb navigation
- [ ] Semantic HTML used
- [ ] Mobile-responsive
- [ ] Fast page load time

## 🚀 Quick Wins

1. **Fix Multiple H1s**: Ensure only one H1 per page
2. **Improve Alt Text**: Make all alt text descriptive and keyword-rich
3. **Optimize Titles**: Ensure all titles are 50-60 characters
4. **Add Internal Links**: Link to related content naturally
5. **Improve Content**: Ensure minimum 300 words, ideally 1000+

  - Performance scores
